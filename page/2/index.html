SEO，按我一贯的划分办法，包括三部分：内容（content）+ 外链（links）+ 技术端优化（technical SEO）。 有一点年龄的老站，一般情况下，发布或更新内容，就能获得不错的排名。 但新站？你会发现，任凭你怎么折腾内容，一般很难排到前几，除非是超级长尾词。 这时候，外链的作用就变得尤其突出，就像老外同行经常说的： You need links to move the needle. 那么，一个新的英文内容站（或者其他类型的网站），该怎么获得第一批高质外链呢？ 今天就来聊聊我所认为（并尝试过）的那些快捷高效的外链获取策略。 首先声明一下，我只谈白帽。 因为外链获取无非以下三种办法： 自己建（build links）：几乎所有类型的提交、博客评论、群发，都属于这个，外链质量差，有些还比较耗时。 去买（buy links）：违反谷歌的外链政策，容易被惩罚，高风险，而且你得花$$$。 吸引（earn links）：这便是我一贯推崇的高质白帽外链，别人愿意给你外链，因为你的网站对他（或他的受众）有价值。 以下聊的便是第三种，也是最有挑战和最有意思的一种，我认为这也是SEO的精髓所在。 HARO（美国一个知名的连接媒体和线人的平台） 适合推广你的主页，运气好的话，能从大媒体站拿到外链，而且你的网站不需要有太多的内容，因为对方要的是你的quote，顺便给你的网站一条外链。 比如，这个月初，我的一个新站，就从Washington Post (DR 93)拿到一条。 怎么操作，看这里： https://ahrefs.com/blog/haro-link-building/ 顺便说一下，因为语言和文化差异，最好让你的写手或者专门招美国人兼职来做这个，可以按结果或时间付费，我是让我的写手们兼职做，按小时付费。 转化率：5%-25%，大概在这个区间，像Matt Diggity这样的高手亲自做能达到25%左右（某次他在Facebook提到），我的写手里最好的能达到10-15%的转化率，也就是说，每回答八九个问题，能有一条被媒体引用，当然，不同的niche，转化率肯定不一样。 Resource Page Link Building（资源类页面） 适合于推广首页或链接资产（linkable assets），网站最好有一定数量的高质内容，还有好的about page。 转化率，我的感觉是1%-5%，当然，取决于你是手工还是自动化，手工挡的话转化率肯定会高些，但代价就是耗时。 怎么操作，看这里： https://backlinko.com/hub/seo/resource-pages 注：手工推广的话，可以结合下面的broken link building策略一起使用，转化率会更好，因为一般情况下对方站长resource page里或多或少会有过时或失效的链接。 Broken Link Building（失效链接） 适合推广某个具体的页面。 常用的做法，先找到你行业里一篇拥有很多外链的文章，但现在文章已经不存在，可能被删、域名没有续费等，然后你撰写一篇类似主题的文章发布到你的网站，再联系所有给失效文章外链的站长们请求外链。 转化率：1%-5%，我感觉跟resource page link building差不多。 怎么操作，看这里： https://ahrefs.com/blog/broken-link-building/ 注：别小看broken link building，可以有更多高级的玩法，取决于你是否足够creative，我最近在跟一个美国的link building agency合作，他就只用这一招，而且你不得不佩服他的执行力。 Skyscraper &amp; Shotgun Skyscraper（摩天大楼法） 适合推广链接资产。 怎么操作，看我之前写的这篇文章。 转化率：Skyscraper （5%-15%），Shotgun Skyscraper（1%-3%） 注：传统的Skyscraper最早由Brian Dean提出，手工操作居多；Shotgun Skyscraper由Authority Hacker团队提出，自动化居多。 The Moving Man Method（“移动人”策略） 适合推广：主页和具体的文章页面。 任何一个行业，总会有竞争对手倒闭或退出，总会有产品停产或更名，他们的网站和产品页面肯定会有一定数量的外链，能不能拿到这些外链，就看你的本事了。 怎么操作，看这里： https://www.theseoproject.org/the-moving-man-method/ 转化率：不定。这个策略我使用的少，转化率没有特别关注。 Link Roundups（行业资讯类） 适合推广：具体的文章页面。 每个行业总会有一些新闻评论者，这些人里面很多有自己的博客（或者是某个杂志版块的编辑），他们会定期的为他们的读者提供过去一天、一周、或者一个月的新闻，而且简短的点评几句。 你的文章能否出现在那里，取决于你能否及时的将你的文章呈现在评论者的眼前。 怎么操作，看这里： https://www.quicksprout.com/how-to-build-links-and-traffic-from-link-roundups/ 转化率：~10%，但这样的机会一般不多，请珍惜去开发。 Guest Posting（客座博客） 适合推广：首页、产品页面、或者具体的文章页。 这里就不多介绍了，想必你也知道，我个人不太喜欢也不常用到这个策略，虽然过去也弄到过不少guest posting的外链，因为过程相对比较冗长，涉及的环节比较多。 怎么操作，看这里： https://ahrefs.com/blog/guest-blogging/ 转化率：因人而异。 创建信息类文章（Informational Articles） 你没看错，这也是一种外链获取策略，而且，是我最喜欢的策略之一。 做好关键词研究是关键，选择那些有搜索量而且是Info类型的词（比如：what is, how to, meaning, etc.），这些词一般是长尾，就算你的网站是新站，自动排上去也不是没可能。 有好的排名后，就会有流量，最重要的是，这些流量里藏着link creators（给你外链的人），他们搜索这个不是为了学习，就是为了找到一个相关的链接加到他们的文章里。 就像我在写作这篇文章时，给了Ahrefs和Backlinko好几个外链，同样的道理。 转化率：没数据。 最后几句 世界上没有最神奇的SEO外链获取策略，一切在于执行和变通，看你是否足够creative，看你是否足够有耐心。 执行的过程中，你肯定会遇到这样或那样的问题和挑战，但，持之以恒，你肯定会有不错的收获。 还有，以上介绍的每一种外链获取策略并非相互排斥，你完全可以融合着使用，比如：当别人拒绝了你的外链请求后，转向Guest Posting，未尝不可。
