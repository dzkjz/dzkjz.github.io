SEO的目的是什么？没错，就是排名！

我这么说，可能很多人会觉得我见识浅薄，俗不可耐，毕竟在很多人心中，SEO可是一门博大精深的学问，怎么可能仅仅是做排名这么简单？

SEO应该是涉及搜索引擎、网络优化、引流、转化等等的一系列学问，其中还包含营销推广等专业知识。没错，这些话应该是无数个SEOer曾经告诉你的，也是你在各种论坛，各种文章中所看到的！

但是，明确告诉你，SEO的最终目的就是网站在搜索引擎上的排名，谷歌百度等搜索引擎的本质是什么，就是根据关键词给网站排名！好的网站排名在前，获得更多展现机会和流量，差的网站排名在后，自生自灭爱咋咋地！所以，你既然是做搜索引擎优化，不优化排名还能优化什么？

但是现在排名有多难做，你能体会吗？像谷歌，一个SEOer想要将一个中等词操作到首页，往往需要半年的努力，还要花费不菲的金钱和精力！这偌大的代价很多公司甚至承受不起！（当然，你要说百度排名好做，我无法反驳，毕竟现在百度有快排这个死穴，但快排毕竟不是长久之计，很多SEOer都认为快排死期将近。）

为何竞争激烈，不是因为你竞争对手增加了，而是因为你的竞争对手选择了跟你一样的关键词！

而所有关键词工具，提供给你的几乎全部都是极其相关词，更恐怖的是，即使你综合使用若干种不同的关键词发掘工具，找到的可以利用的关键词其实也没有什么区别。

很明显，这就是为什么做同一产品的网站使用的关键词几乎一模一样的原因。

  1. 有些词在阿里有搜索量，在谷歌完全没有，有些词在阿里没人用，在谷歌里却是非常热的词

  2. 阿里、MIC等是B2B平台，这些平台上所有卖家都是批发商、生产商，所以用户在这个平台上搜索不需要加supplier、manufacturer这些，单纯搜产品词就可以了，反正找到的肯定是批发商。（这个也是解释为什么B2B不能直接拿产品词当主词？）

    • 中心关键词一般是一到三个，当然一个的话最好操作，三个的话就特别麻烦（因为还是需要给它们分清主次，不然三个词都搞不好），至于再多，那肯定就不是中心词了

    • 中心词如何确定，拿产品或者产品分类页面来说，叫法准确、搜索量高的词就是中心词（机械类、工业材料类产品一般如此，某些偏零售的产品可能必须加supplier、manufacturer这种属性词才行）

    • 这里举个例子，比如我们是做测试设备的，某个分类是纺织测试设备，英文名称是textile testing equipment，但是我们通过分析竞争对手网站和使用关键词工具（GKP、kwfinder等等）查询，发现这个词还有几个近似的叫法，分别为textile testing instruments和textile testing machine以及textile tester，通过考察分析，基本可以确定，搜索这些词的客户找的都是同一类产品。然后看查看搜索量：

      （textile tester月搜索量只有10，这里不列出来了）

      所以，如果中心词只有一个那肯定是textile testing Instruments（搜索量最高且难度最低），如果是三个，那依次就是textile testing Instruments、textile testing equipment、textile testing machine。

    • 合不合适，至于如何判断合不合适，标准有很多。但是一般来说都比较通用，还是举实际例子说明，例如，地区词：

      然后还有一种，我们可以称之为精准产品词（形容词+产品词）：像这种词，你可以考虑覆盖，也可以考虑用更精确的产品来覆盖，一般对于这种机械类产品来说，分类词搜索量是最大的，到了单个的产品词其实搜索量很小甚至没有，所以这个时候，很多分类覆盖不过来的长尾词，其实我们完全可以考虑用放到单个产品里面。【就是说 产品页面里面放置这些产品词。】

    通过上面三个标准，我们可以把相关长尾词好好的排除一下，但是依然会剩下一批（可能20个以上）。对于这一批词，我们应该如何布局呢。

  3. 长尾词如何布局

    1. 首先，最重要的词才可以出现在标题中，并且越重要就越要在标题前面。（如果按照这个例子，标题应该是textile testing Instruments, textile testing equipment, textile testing machine或者是textile testing Instruments,equipment,machine supplier & manufacturer等等，具体如何来做，需要更深的实践经验，考虑更多东西，一般情况这两种就比较合格了。）
    2. 其次，最重要的词出现在最前面的文字内容中，毫无疑问textile testing Instruments, textile testing equipment, textile testing machine这些词都要尽可能出现在页面内容第一段的，而且在第一段越前面的位置越好，其他的长尾词，按照重要程度也是尽量的往前放，越往前，越容易获得排名。
    3. 第三，关键词能连在一起最好，但是为了用户体验（语句自然程度、通顺程度），很多词可以分开，简单来说一句话，textile XX testing Instruments XX XX manufacturer也算把textile testing Instruments覆盖了，只是没有连在一起效果那么好，但是一般情况下，不重要的词完全可以用这种断开的形式覆盖。
    4. 第四，最重要的词，在H标签里出现一下，比如这种词textile testing Instruments，如果可以的话，可以让它在H标签里出现一下，但是注意一定要自然。
    5. 第五，最重要的几个词，使用黑体、斜体、下划线之类的标注一下。
    6. 第六，最重要的关键词在内容结尾也出现一下。
    7. 第七，图片标题和alt标签可以适当出现一下关键词。
    8. 第八，不要在乎什么关键词密度，像那种不重要的关键词，只要在内容里出现一遍就可以了，哪怕不连在一起也没事，重要的词只要保证出现在内容靠前的位置，最重要的词出现在标题、内容最前方、H标签、末尾，这样就足够了，根本不需要刻意增加关键词密度。
    9. 第九，内容思路要清晰，文字要自然，要注意用户体验（重要的地方加黑，用一下其他颜色标注下，加一下内链等等），千万不要为了覆盖关键词瞎写。

优质英文产品描述怎么写

对于低竞争产品，网站文字内容几乎决定一切。

  1. 文字内容的要求，

    主要有这么几点：

    1. 首先，不能有语法和拼写错误，这是基础。
    2. 其次，内容需要原创，而且每个页面的内容重复度要尽可能低。
    3. 最后，内容越长越好，涵盖的相关话题越丰富越好，当然更重要的是有价值，要让客户看了觉得有用，觉得很nice。
    4. 对于大部分B2B和B2C商品站来说，网站文字最重要的部分就是产品描述！

  2. 但是很多网站，产品描述基本上是应付事：

    1. 很多是从阿里或同行网站把文字原封不动搬过来的；
    2. 还有些就随便弄了个参数表格，连像样的文字介绍都没有；
    3. 好一点的，文字介绍是让业务员写了不少，但是各个页面重复度很高，很多产品都是用的极其类似的介绍，

    类似这些情况，网站内容都是不及格的，拿这样的内容来做SEO，想上排名难度太大！

  3. 那产品描述到底应该如何写呢？

    1. 首先，你要确定客户需求

      客户搜索产品词和产品相关词（长尾词、LSI词）的时候，他们都是有一定的需求的，他们的需求包含哪些方面？我们应该如何解决他们的这些需求？

      1. 一般来说，仅仅依靠主关键词，我们只能主观判断客户需求，
      2. 但是通过长尾词和LSI词，我们能更客观的了解客户的搜索目的

      拿机械类产品来举个栗子吧：transformer winding machine（这是我随便搜索的一个纺织类产品，我也不知道是什么产品，后续如果有解释错误的，行家别见怪）

      单看这个词，我们可以想象，客户的需求主要有两个方面：

      1. 想了解transformer winding machine是什么、有什么用？

      2. 想搜索相关厂家和代理商进行购买。

      但是如果我们仔细了解下这个词的相关词（使用Ubersuggest/keywordtool/Lsigraph工具查找），下面是通过keywordtool.io搜索到的部分相关词：

      通过浏览keywordtool显示的全部的长尾词以及查看lsigraph.com提供的LSI词，可以发现：客户更进一步的搜索需求主要包含这个几个方面：

      1. transformer winding machine+地区（这个需求可能有几种，一种是外地客户想购买特定地方生产的这类机器，比如，美国人想购买印度生产的这类机器：另一种是本地客户想购买本地的机械，例如巴基斯坦人想购买巴基斯坦本地的这类机器）。

      2. transformer winding machine price（想了解价格）。

      3. diy和how to build transformer winding machine（想了解如何自己制造或者说组装这类机器）。

      4. transformer winding machine的相关产品coil winding machine。（这种一般是两个词是一类产品的两种叫法，或者两类产品非常相近、互相包含等，因为有人会经常一起搜索，或者搜索不同的词进入同样的网页，因此谷歌将之归为一类词。）

      5. transformer winding machine和coil winding machine更细小的一些产品分类（比如transformer winding machine automatic、electric motor winder machine等，这种很明显，用户想要满足更精细功能的这类产品）。

    2. 布局内容

      理解完了需求，我们在产品介绍的时候很容易就知道怎么写了。比如，如果要写transformer winding machine的产品介绍，根据需求，我会这样来布局内容：

      1. 首先，简单介绍下transformer winding machine是什么，有什么用途，如果有别名的话，一起包含进去（就说某某机械也被称为某某机械，一句话带过就行）。

        介绍差不多了，丢一句话，想详细了解的可以看维基百科，留一条链接（为什么要留链接，因为做谷歌排名，网页有导出到高权重网站的链接是有好处的！）。

        这样，这些内容可以完全满足搜索需求是为了解信息的客户了。（这样一段话相信怎么着也得100词吧）

      2. 接下来，介绍我们企业生产的这类产品的特点，款式、跟同行对比有什么优势，采用了什么全新的技术，使用了什么原料，质量如何，有哪些地方做了改进，属于世界第几流，跟哪些著名品牌有一拼等等，最后，详细参数。

        （这个主要是为了吸引客户，实实在在写，尽量不要吹，这段内容相信怎么也有150词，这是营销自己的最重要内容，写不了150词可以GG了。）

      3. 然后是价格，上面质量说完了，客户肯定会对价格有个预期，但是直接说价格肯定也是不好的，而且那些搜索价格的客户肯定也是想了解整个这类产品全世界、高中低、不同生产商的所有价格，以便做到心里有个数。

        所以，写价格我们要这么写，首先最高的，先把几款国际知名品牌价格写出来（口说无凭，最好有截图或者引用链接），然后评判一下这些价格（如何评判？客观的说下谁会选择这些产品就可以啊，比如说国企，超大型企业等。然后分析整体价格里面，质量占多少分、品牌价值占多少分、渠道占多少分、广告占多少分。）

        接下来，中档的也是这样介绍，低档的也是同样套路，当然别忘了自己的产品，介绍清楚自己属于哪一档，如果价格低，写清楚为什么低，如果高，同样写清楚为什么高！这样，基本想了解价格的客户需求解决了，而且也适当营销了自己，说不定就能给转化了。（这个也能写100词吧。）

        （当然，对于价格，其实列一个详细的产品价格调研清单，然后列明出处，再做一下评论，这个是很好的，如果推出去，有可能能引发病毒营销，并带来外链。）

      4. 然后就是产地了，产品+产地是非常容易出现的，客户的搜索需求往往也是上面列出的两种，如何满足这两种需求呢，很简单。

        • 第一种，外地客户想购买特定地区的产品，为什么？无非就是因为某个地区专门生产这个，价格低或者质量高呗，如果我们不是这个地区怎么办？建议可以写我们工厂跟这个地区主要工厂有合作关系，当然你可以说是货源地，或者原来在这里来后来搬迁了等等，最重要的要点出，你们的价格和质量跟他们相比差不多，或者更优秀。当然，不管怎么说得有一定根据才行，不能胡扯。

        • 第二种，本地客户就想找本地的供应商，你怎么办呢？也很简单啊，你可以说某某地是你最大的市场，百分之八十的客户都是来自这里（此处最好有案例），是你最重要的出货地区，因为出货量大，你在这些地区建立了完善的售后和服务体系，甚至投资建立了生产基地等等。还是那句话，可以说，但是要有根据。

          这样地区需求也解决了，至少也得150词吧。

      5. 再然后，diy的部分，有些人就是不想买想自己制作，或者可能是买了产品想自己组装，对于这些客户，我们同样要好好对待，如果你能写好的教程，那一定详详细细讲出来，如果没有，你可以百度或者谷歌出来，然后放一条链接。这里，有的同学可能在想，我写不出教程来，也不想放别的网站的链接，那OK，这段完全别写！如果你写了，却解决不了用户需求，只能给客户留下这样的印象：“标题党，把我骗来，就说一堆废话。”这样适得其反。（这段话也能有100词左右，如果内容很多，可以写一篇专门文章，这里放一条链接即可。）

      6. 最后，是不同产品或者这类产品更细分产品的需求，这个其实也很好写，对于不同产品，你可以直接写区别啊，这两种产品有什么异同，如果都是一种用途，那各有什么优劣势，如果内容很多，在这里不方便展开，你可以单独写一篇文章，然后在这里放一条链接连接到文章即可。至于更细分的产品也是这样，详细的讲一下就可以了，展不开就放链接。这里也得有150词左右。

      7. 上面这些是我们通过关键词分析出来的需求，但是作为一个老练的业务员，我们是很明白客户找到我们会有哪些其余的疑惑以及想了解的问题，其实我们可以一并放在这里的。

        比如客户会关心配送、担保、售后等等，这些客户经常问的问题，你写在产品介绍里不很好吗，弄上200词，既能增加字数，稀释重复度，又能促进转化，何乐而不为？

        当然内容不仅是在解决需求，同时也是在营销，所以，写这些内容的时候你要穿插着把企业优势介绍一下，让自己的品牌多出现一下，尤其是全部介绍完的时候，再来个50词的产品或者品牌介绍。

        同时，对于关键词也是一样，像做2B，常见的必须加的词supplier、manufacturer、factory、company、oem、wholesale、bulk、cheap、china等等，这些很自然要全部出现一到两遍。其他的用工具搜到的，比较重要的关键词更不用说，该放到H标签的放H标签，改黑体的黑体，改斜体的斜体，下划线什么的也用下，重要的想要客户注意的，不同颜色标注下也是应该的。

      写到这里，可能很多同学深受启发，还有一些同学感觉，我的产品差异太小了，就算按照这个写也写不了这么多内容，或者写出来重复度也还是很高。

      是，的确也有情况，如果遇见那种产品及其相似的，有两个办法：

      1. 使用canonical标签，适合2C站。
      2. 将极其类似的产品统统放到一个页面，比如有十个产品及其类似，几乎没什么区别，那你干嘛放一个页面，统统放到一个页面去，当然，这个肯定是适合2B站。

      另外，关于内容，每个句子尽量简短易懂，每段话也是尽量短一些，层次分明一些，版式也清爽一些，能有图表的用下图表，有相应图片和视频就都加下，这样不管是客户还是搜索引擎感觉都会很好。

这些死链如何处理？

第一种是301重定向，做了301之后，用户访问原来URL直接跳转到新URL。这种方法其实很好，因为301可以保证原先页面的权重不流失，并且如果原先页面有很多的访问，跳转到新页面之后，这些用户也不会流失。

如果全部做301，必须保证原网页和新网页内容基本一致。

第二种，如果没做重定向，只是单纯删除页面，那一定记得要把被删页面的URL提交谷歌，提交方式很简单，到这里：https://www.google.com/webmasters/tools/removals，输入删除URL，点击提交就可以了。

​ 但是，被人忽略的一个问题就出现了——内链问题，想一想某条内链指向的页面被删掉了，然后这个页面提交了谷歌，在谷歌的搜索结果确实清理掉了，但是网站内部，访问这条内链指向的仍然是不存在的页面，这条内链仍然是死链，如果做重定向还稍微好点，但肯定不如内链直接指向重定向后的页面。

所以，如果想彻底解决死链问题，内链死链要么彻底删掉，要么直接修改成重定向之后的页面。

如何判断外链质量（或者是什么是高质量外链）

  1. 是不是自然的外链？

    什么是模拟自然？简单说就是，全面想象一下自然产生的外链有什么特点，然后做符合这些特点的外链，这里没法详细讲，只能说内容营销outreach就是最大程度的模拟自然。

  2. 相关性越高，外链质量越高，同时相关性又分两块：内容相关和网站相关。

  3. 是否来自真实网站？什么是真实网站？举个反例，你注册的0博客，内容全是采集复制伪原创，根本就没想给人看，只是为了给目标网站导外链，这就不能称之为真实网站，PBN也一样，内容全是垃圾，唯一目的就是外链，这种就是典型假网站。所以，网站做出来真正给人看的，为了传播知识、营销品牌、产品销售等等的都是真网站。

  4. 看网站流量数据（ahrefs的关键词数量、预估自然流量、预估自然流量价值），排除0，最后看网站的DA、PA、DR、UR，这些当然是越高越好。
